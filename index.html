Onde tudo começou?

HTML

O HTML é uma Linguagem de Marcação de Hiper Texto surgiu em função da necessidade de criar artigos que pudessem ser visualizados atravé de um computador, sendo o HTML a linguagem de marcação que estrutura o artigo e que também referencia outros artigos através de links já que esse era o principal propósito dessa linguagem, já que o inventor Tim Berners Lee tinha por objetivo criar esses artigos e disponibilizar online e que também esses artigos pudessem conter links que referenciassem citações para o artigo original

HTTP

O HTTP é um Protocolo de Tranferência de Hiper Texto foi criado para que fosse possível criar um canal de comunicação padronizado através das redes que conectam os computadores, dessa maneira foi criado o HTTP para convencionar como a transferência dos arquivos HTML iria acontecer através das redes para que os computadores pudessem reconhecer esse padrão.

Browser

O Browser ou Navegador foi criado para que o usuário tivesse uma plataforma para acessar esses artigos que hoje conhecemos por websites que são formados por webpages. Os navegadores são capazes de interpretar a Linguagem de Marcação (HTML) e estruturar o site com essas informações que foram interpretadas. Nos dias de hoje os navegadores também interpretam outras linguagens como o CSS (Cascade Style Sheet ou Folhas de Estilo em Cascata) que são responsáveis por dar estilo as páginas através de fontes de letras diferentes, cores, tipos de fonte, tamanho, posicionamento na página entre outras funções e os Navegadores também interpretam a linguagem Javascript que é reponsável por dar função aos elementos da página, exibir animações entre outras funcionalidades.

URL

O termo URL que é um Localizador Uniforme de Recursos padroniza uma série de informações específicas explicadas abaixo:

Exemplo de URL: Protocolo://Nome_do_Dominio:porta/caminho_do_Arquivo?parâmetro=informacao#âncora

Protocolos

  • Https: porta 443
  • Http: porta 80
  • ftp: porta 21
  • file: local

Tags HTML

A maioria das tags HTML possuem tags de abertura e de fechamento, porém também existem algumas tags que não possuem tag de fechamento

As tags também possuem atributos onde podemos editar estilo, utilizar identificadores de classe e id e também informações e funções

A tag !DOCTYPE HTML indica que esta página é um documento HTML e deve ser interpretada desta maneira pelo navegador

A tag HTML delimita o inicio e o fim do documento HTML

A tag HEAD é onde ficam as configurações de metadados como os explicados abaixo:

  • A tag TITLE é utilizada para definir o título do site na aba de navegação, ao adicionar o site aos favoritos e também como indexador utilizado como referência pelos buscadores
  • Através de uma tag LINK podemos adicionar um FAVICON que é uma imagem que aparece como um icone ao lado do titulo na aba de navegação e também quando o usuário salva o site como favorito criando estilo e referencia de marca
  • A tag META com o atributo CHARSET define o conjunto de caracteres que será interpretado pelo navegador
  • A tag META possui diversos atributos que podem ser conferidos neste LINK
  • A tag META também possui atributos para redes sociais, conhecidas como SOCIAL META TAGS
  • A tag STYLE é utilizada quando se quer utilizar estilização em CSS no próprio documento HTML
  • A tag SCRIPT é utilizada quando se quer utilizar código Javascript para manipular os elementos da página HTML através do DOM
  • A tag BASE é utilizada com o atributo HREF para configurar uma URL que serve de base para concatenar com o HREF de links exibidos nesta página que direcionam para a página indicada após a concatenação

A tag BODY é onde ficam as tags que formam a estrutura e o conteúdo do site que será exibido no navegador

A tag P é utilizada para criar parágrafos

A tag HEADER é utilizada para identificar a sessão onde ficará o cabeçalho do site

A tag FOOTER é utilizada para identificar a sessão onde ficará o rodapé do site

A tag NAV é utilizada para identificar o menu do site

A tag SPAN é utilizada para editar ou estilizar uma parta especifica de um parágrafo ou de um título

A tag SECTION é utilizada para identificar uma sessão do site

A tag DIV é utilizada para identifiar uma divisão do site, que seriam subdivisoes de uma sessão

A tag U significa underline e é utilizada para sublinhar o texto

A tag B significa bold e é reconhecida pelo navegador como uma marcação de Negrito que deixa o texto mais destacado

A tag STRONG significa forte e é reconhecida pelo navegador como uma marcação de Negrito também, mas também é reconhecida pelo leitor para cegos e dá mais entonação na sua fala

A tag I significa itálico e é reconhecida pelo navegador como uma marcação de Itálico que realça o texto com uma caligrafia estilizada

A tag EM significa Emphasis (ênfase) é reconhecida pelo navegador como uma marcação de Itálico que realça o texto com um caligrafia estilizada e também é reconhecida pelo leitor para cegos e dá mais enfase na sua fala

A tag Q significa quotes (citação) e é reconhecida pelo navegador como aspas duplas que demarcam a citação de um autor que se deseja referenciar

A tag UL é utilizada para apresentar uma lista não ordenada e a tag OL é utilizada para apresentar uma lista ordenada, e a tag LI são os itens da lista seguem os exemplos abaixo

Lista não ordenada:

  • Primeiro Item
  • Segundo Item
  • Terceito Item

Lista ordenada:

  1. Primeiro Item
  2. Segundo Item
  3. Terceiro Item

A tag BUTTON é utilizada para criar um botão que posteriormente pode receber uma funcionalidade através de um código Javascript

A tag A é utilizada para criar um link que referência outra página através do atributo HREF, onde é possível passar uma URL da página que será o destino do link

Através da tag A também podemos utilizar ÂNCORAS para fazer um scroll para uma determinada seção da própria ou de uma outra página, onde identificamos a sessão com uma tag ID e passamos como informação para o atributo HREF do link o nome do id da seção precedido pelo caractere # (Exemplo href="#id")

Através da tag A também podemos utilizar o atributo HREF para encaminhar o link para abrir um aplicativo/software de email ou para sincronizar com seu celular sendo possível fazer uma ligação, conforme nos exemplos a seguir:

href="mailto:email@email.com"

href="tel:+48999999999"

Quando for utilizar os atributos HREF de links ou SRC de imagens, videos entre outros componentes é muito importante referenciar os arquivos utilizando o caminho relativo para navegar entre os diretórios do site, pois assim não se corre o risco de perder a referência.

A tag IMG é utilizada para adicionar imagens ao documento html, e os atributos SRC que indica o local da imagem e o atributo ALT que adiciona uma descrição da foto para que o leitor de tela possa fazer a leitura para deficientes visuais são obrigatórios

A tag VIDEO é utilizada para adicionar um video a página HTML, sendo o atributo SRC que indica o local do arquivo de video sendo ele local ou em um servidor na web obrigatório.

A tag IFRAME é utilizada para adicionar videos do youtube a página HTML sendo possível incorporar o código do vídeo na página

Tabelas

As tabelas em html são criadas utilizando as tags abaixo:

  • A tag <table> é utilizada para delimitar o inicio e final da tabela
  • A tag <th> é utilizada para criar um titulo definindo que tipo de informação a coluna irá conter
  • A tag <tr> é utilizada para criar uma linha na tabela
  • A tag <td> é utilizada para criar uma coluna dentro da linha
  • O atributo rowspan utilizado nas tags <td> são utilizados para indicar quantas linhas uma célula da tabela irá ocupar
  • O atributo colspan utilizado nas tags <td> são utilizados para indicar quantas colunas uma célula da tabela irá ocupar
  • A tag <thead> é utilizada para delimitar o cabeçalho da tabela onde vão os titulos das colunas
  • A tag <tbody> é utilizada para delimitar o corpo da tabela onde vão as informações
  • A tag <tfoot> é utilizada para delimitar o rodapé da tabela

Estrutura Hierárquica de um Documento HTML

Importante mostrar a estrutura hierárquica de uma página HTML, para poder entender como funciona e poder fazer a construção ou manutenção da página de forma correta sem quebrar a estrutura da página. Confira a imagem abaixo:

DOM

DOM: Document Object Model é uma representação da página HTML no navegador que pode ser modificada no tempo de execucação, podemos acessar o DOM através da tecla de atalho F12

GetElementById: é utilizado para buscar um elemento HTML pelo seu identificador ID e após localizar é possível realizar alterações neste elemento em tempo de execução conforme vimos anteriormente na explicação do DOM

document.getElementById("id").value = 'Olá Mundo'

GetElementsByClassName: é utilizado para buscar um ou mais elementos pelo seu identificador de Classe e após localizar estes elementos é possível realizar alterações em tempo de execução.

document.getElementsByClassName("class").value = 'Olá Mundo'

GetElementsByTagName: é utilizado para buscar um ou mais elementos pelo seu identificador de Tag e após localizar estes elementos é possível realizar alterações em tempo de execução.

document.getElementByTagName("tag").value = 'Olá Mundo'

De acordo com o objeto selecionado é possível verificar qual é seu elemento pai, filho(s) e também buscar um filho em específico conforme os comandos abaixo:

  • ParentElement: identifica o elemento pai do elemento referenciado
  • Children: identifica o elemento filho do elemento referenciado
  • FirstElementChild: identifica o primeiro elemento filho do elemento referenciado
  • LastElementChild: identifica o último elemento filho do elemento referenciado
